\begin{zremark}
  在线性方程组有解的前提下，是否可能存在某个解，它不是通解中的一个特例？
  换句话说：通解是否包含了所有解？
\end{zremark}

\textbf{证明：}

一般线性方程组，都可以通过一个特解加基础解系的线性组合来表示所有解。

\begin{zremark}
  证明向量组线性无关的常用办法。
\end{zremark}

设
\begin{align*}
  \alpha_1, \alpha_2, \cdots, \alpha_n \ \ \ (I)
\end{align*}

\begin{itemize}
  \item 借助定义，证明:$k_1\alpha_1 + k_1\alpha_2 + \cdots + k_n\alpha_n$可得
        \begin{align*}
          k_1 = k_2 = \cdots = k_n = 0
        \end{align*}

  \item 借助一些已经存在的线性无关的向量组。比如：基础解系中的向量。
        如：第二章$\S 4$习题12。
\end{itemize}
